getInputAmount()​

getInputAmount(outputAmount, sqrtPriceLimitX96?): Promise<[CurrencyAmount<Currency>, Pool]>

Defined in: entities/pool.ts:257

Given a desired output amount of a currency, return the computed input amount and a pool with state updated after the trade Works only for vanilla hookless v3 pools, otherwise throws an error

Parameters​

ParameterTypeDescription
outputAmountCurrencyAmount<Currency>the output amount for which to quote the input amount
sqrtPriceLimitX96?JSBIThe Q64.96 sqrt price limit. If zero for one, the price cannot be less than this value after the swap. If one for zero, the price cannot be greater than this value after the swap

Returns​

Promise<[CurrencyAmount<Currency>, Pool]>
