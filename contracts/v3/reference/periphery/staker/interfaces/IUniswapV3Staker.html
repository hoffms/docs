    bytes32 incentiveId
  ) external view returns (uint256 totalRewardUnclaimed, uint160 totalSecondsClaimedX128, uint96 numberOfStakes)

Parameters:​

NameTypeDescription

Return Values:​

NameTypeDescription

totalSecondsClaimedX128uint160Total liquidity-seconds claimed, represented as a UQ32.128
numberOfStakesuint96The count of deposits that are currently staked for the incentive
